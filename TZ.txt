Добавить возможность создания фильма используя CBV

1. Создать форму для создания фильма
    Форма должна отправляться методом POST на view для создания фильма
2. Добавить кнопку создания фильма в шаблоне фильмов
    Кнопка должна перенаправлять на страницу создания фильма
3. Добавить метод создания в view для фильмов
    a) Получить данные из формы
    b) Создать фильм
    c) Перенаправить на страницу фильма
